## 📝 Sobre o Projeto
Este é um utilitário desenvolvido em **Python** para ajudar usuários a ter uma melhor visualização de um reservatório de água, destacando cada situação com uma cor específica.

### ✨ Diferenciais técnicos:
- **COLORAMA:** Utilização da biblioteca para auxiliar na efetiva visualização mais clara.


---

## 🧮 Fórmulas
O sistema utiliza a fórmula de cores para definir os niveis.
ex:
nivel 1 - Muito baixo (critico) será exibido em vermelho
ENQUANTO nivel 5 - Muito alto (alerta) será exibido em azul
